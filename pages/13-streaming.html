13. 動画ストリーミング・OneDrive・Stream

← トップ
HLS / DASH マニフェスト、Microsoft Stream、OneDrive のメディア共有 URL、YouTube の ?itag= パラメータなど、検出器がストリーミングコンテンツとして認識すべき URL のセット。

ローカルマニフェスト（実体ファイル）

  • sample.m3u8 (HLS)
  • sample.mpd (DASH)
  • sample.pls (Playlist)
  • sample.xspf (XSPF)

HLS / DASH マニフェスト URL（外部）

  • /video/master.m3u8
  • /sd/index.m3u8
  • /video/manifest.mpd
  • .mpd?token=...

HLS/DASH パスベース検出（マニフェスト名なし）

  • /hls/<path>
  • /dash/<path>
  • Smooth Streaming /Fragments(...)
  • /QualityLevels(...)/Fragments(...)

Microsoft Stream / OneDrive

  • microsoftstream.com/video/<id>
  • stream.office.com/.../video/<id>
  • 1drv.ms/v/ (動画共有)
  • 1drv.ms/i/ (画像共有)
  • 1drv.ms/w/ (Word 共有)
  • 1drv.ms/x/ (Excel 共有)
  • 1drv.ms/p/ (PowerPoint 共有)

YouTube / 動画ストリーム特殊パラメータ

  • YouTube ?itag=18
  • ?itag=251 (audio)
  • ?content_type=video/mp4
  • ?mime=audio/mp3
  • ?content_type=image/png

video / audio タグでマニフェスト指定

video.js などで m3u8 を直接 src に指定するパターン:

動画ID系クエリ（業務動画ポータル想定）

  • ?videoId=<id>
  • ?movieId=<id>
  • ?streamId=<id>
  • /video/play/<id>
  • /stream/video/<id>
  • /media/video/<id>
  • /video_browse/<id>
  • /video_view/<id>
